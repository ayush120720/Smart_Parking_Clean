# 🚗 Smart Parking System

An intelligent parking management system built using Spring Boot that allows users to book parking slots, manage reservations, and handle payments efficiently.

---

## 📌 Features

- 🔍 **Real-time Slot Availability**
- 📅 **Slot Booking System**
- 👤 **User Registration & Authentication**
- 💳 **Payment Module (UPI / Card / Cash)**
- 🧾 **Receipt Generation (PDF)**
- 📊 **Dynamic Slot Management using DTO**
- 🔄 **Reservation Management**
- 🌐 **Frontend using Thymeleaf**

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot
- **Frontend:** Thymeleaf, HTML, CSS
- **Database:** MySQL / H2
- **Architecture:** MVC (Controller-Service-Repository)
- **Other Tools:** iText (PDF generation)

## 🚀 How It Works

1. User registers and logs in  
2. System shows available parking slots  
3. User selects slot and booking duration  
4. System calculates amount (duration × 100)  
5. User completes payment (UPI / Card / Cash)  
6. Booking is confirmed and receipt is generated
